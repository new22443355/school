# Инструкция по подключению Яндекс.Метрики

## Шаги для активации счетчика:

1. **Зарегистрируйтесь на Яндекс.Метрике:**
   - Перейдите на https://metrika.yandex.ru
   - Войдите через Яндекс аккаунт
   - Нажмите "Добавить счетчик"

2. **Создайте счетчик:**
   - Укажите адрес сайта: https://new22443355.github.io/school
   - Выберите опции: Вебвизор, Карта кликов, Отслеживание хэша
   - Нажмите "Создать счетчик"

3. **Получите номер счетчика:**
   - После создания вы получите номер (например: 12345678)
   - Скопируйте этот номер

4. **Активируйте код на сайте:**
   - Откройте файл `index.html`
   - Найдите строки с комментарием `<!-- Яндекс.Метрика -->`
   - Удалите символы `<!--` в начале и `-->` в конце блока
   - Замените `XXXXXXXXX` на ваш номер счетчика (2 места!)

5. **Примените к другим страницам:**
   - Скопируйте активированный код
   - Вставьте в `<head>` файлов: about.html, portfolio.html, faq.html

## Проверка работы:

- После активации зайдите на сайт
- В течение 15 минут в Метрике появится первый визит
- Проверьте отчет "Онлайн" в Метрике

## Альтернатива - Google Analytics:

Если предпочитаете Google Analytics:
1. Создайте аккаунт на https://analytics.google.com
2. Получите код отслеживания (gtag.js)
3. Вставьте в `<head>` всех страниц

## Что отслеживается:

✅ Количество посетителей
✅ Источники трафика
✅ Популярные страницы
✅ Время на сайте
✅ Конверсии (отправка формы)
✅ Карта кликов (куда кликают пользователи)
✅ Запись сеансов (Вебвизор)
